### Setting Up Git On Linux

```
git config --global user.name "username"

```

```
git config --global user.email "email"

```

```
ssh-keygen -t ed25519 -C "email"

```

```
eval "$(ssh-agent -s)"

```

```
ssh-add ~/.ssh/theprivatekey

```

Test if its working
```
ssh -T git@github.com

```
